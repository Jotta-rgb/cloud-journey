#  aws-cloud-journey

##  Sobre
Este repositório foi criado para documentar toda a minha evolução, estudos teóricos e laboratórios práticos na Amazon Web Services (AWS), a principal plataforma de nuvem do mercado.

##  Objetivo
Registrar o aprendizado sobre a infraestrutura global da AWS, seus principais serviços, boas práticas de arquitetura (Well-Architected Framework) e a preparação para certificações.

##  Conteúdo Programado
- `fundamentos`: Conceitos básicos de Cloud, precificação e infraestrutura global.
- `servicos-aws`: Estudos práticos focados em EC2, S3, RDS, VPC, IAM, etc.
- `laboratorios`: Projetos e arquiteturas práticas montadas no console ou via CLI.
- `anotacoes-certificacao`: Resumos e simulados focados nas certificações da trilha Cloud.

##  Ferramentas Utilizadas
- AWS Console / AWS CLI
- Git & GitHub
- Markdown para documentação
